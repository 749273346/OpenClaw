# 惠电宝 (Huidianbao) - 功能清单 (Feature List)

| 模块 (Module) | 功能点 (Feature) | 描述 (Description) | 状态 (Status) |
| :--- | :--- | :--- | :--- |
| **核心对话** | **多轮对话** | 支持上下文记忆，能够基于之前的对话内容进行连续追问。 | ✅ 已上线 |
| | **意图识别** | 自动识别用户意图（闲聊、查规程、看图、查状态），分流至不同技能。 | ✅ 已上线 |
| | **模型路由** | 根据问题复杂度自动切换 DeepSeek-Chat (快速) 或 DeepSeek-Reasoner (深度推理)。 | ✅ 已上线 |
| **知识检索 (RAG)** | **混合检索** | 结合关键词 (BM25) 与语义 (Embedding) 检索，提高召回率与准确性。 | ✅ 已上线 |
| | **精准溯源** | 每一条回答均标注来源文件及条款号（如《安规》第XX条），支持点击跳转。 | ✅ 已上线 |
| | **本地知识库** | 支持 Markdown 格式的本地规程文档索引与更新，无需上传公有云。 | ✅ 已上线 |
| **多模态能力** | **图像识别** | 集成 Zhipu GLM-4V，支持上传设备照片进行缺陷识别与场景分析。 | ✅ 已上线 |
| | **语音交互** | 支持接收企业微信语音消息，转文字后处理并可语音回复（需配置）。 | ✅ 已上线 |
| **渠道集成** | **企业微信 (WeCom)** | 深度集成企业微信应用，支持单聊、群聊、媒体消息。 | ✅ 已上线 |
| | **飞书/钉钉** | 支持飞书与钉钉的基础消息收发（备用渠道）。 | ⚠️ Beta |
| **运维监控** | **系统状态监控** | 实时监控 CPU、内存、进程状态，通过 `system-monitor` 技能查询。 | ✅ 已上线 |
| | **自动恢复** | PM2 进程守护，支持崩溃自动重启与开机自启。 | ✅ 已上线 |
| | **热更新** | 支持知识库索引的增量/全量构建，无需停机即可更新知识。 | ✅ 已上线 |
| **开发与扩展** | **技能插件化** | 基于 OpenClaw 技能架构，支持 Python/Node.js 快速开发新技能。 | ✅ 已上线 |
| | **安全沙箱** | 代码执行在受限沙箱环境中，保障系统安全。 | ✅ 已上线 |

## 规格参数 (Specifications)
- **响应时间**: 简单查询 < 2s，复杂推理 < 15s
- **并发支持**: 单实例支持 50+ QPS (取决于底层 LLM 配额)
- **知识库容量**: 支持百万级字符的规程文档索引
- **部署环境**: Linux (Ubuntu 22.04+), Python 3.10+, Node.js 18+
