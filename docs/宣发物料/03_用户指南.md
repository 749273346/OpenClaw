# 惠电宝 (Huidianbao) - 用户操作指南 (User Guide)

## 1. 快速开始 (Quick Start)

### 1.1 关注与绑定
1. 在企业微信中找到“惠电宝”应用。
2. 点击进入应用，发送“你好”或“开始”激活助手。
3. 收到欢迎语“你好！我是惠电宝，您的电力线路工专属知识库助手”即表示连接成功。

### 1.2 第一次提问
直接在对话框输入您的问题，例如：
> “变压器巡检需要注意什么？”
> “倒闸操作的五防是什么？”

惠电宝会稍作思考（显示“惠电宝正在思考...”），然后给出基于规程的详细解答。

## 2. 核心功能使用 (Core Features)

### 2.1 查规程 (Regulatory Query)
支持模糊查询、场景化提问。
- **示例**: “绝缘手套多久检验一次？”
- **技巧**: 尽量描述清楚具体的设备名称或作业场景，有助于提高答案准确度。

### 2.2 看图识隐患 (Image Analysis)
1. 点击输入框旁的 `+` 号，选择“拍摄”或“相册”。
2. 上传一张现场设备照片（如变压器、绝缘子、开关柜）。
3. 发送图片后，可附带文字说明，如“帮我看看这个绝缘子有没有问题”。
4. 惠电宝将分析图片内容并反馈结果。

### 2.3 语音提问 (Voice Interaction)
1. 按住说话按钮，直接说出您的问题。
2. 松开手指发送。
3. 惠电宝会自动将语音转为文字并进行回复。

## 3. 常见问题 (FAQ)

**Q: 惠电宝的回答准确吗？**
A: 惠电宝的回答主要基于内置的铁路电力安全规程与管理规则。虽然我们采用了先进的检索技术，但 AI 可能存在幻觉，**请务必以回答中引用的规程原文为准**，关键作业请二次核实。

**Q: 我可以上传涉密文档给惠电宝分析吗？**
A: **严禁上传**涉及国家秘密或企业核心机密的文件与图片。虽然系统部署在内网/私有环境，但请遵守保密规定。

**Q: 遇到“系统繁忙”或无响应怎么办？**
A: 请稍后重试。如果持续无响应，请联系系统管理员（运维团队）检查后台服务状态。
