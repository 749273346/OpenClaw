# 惠电宝 (Huidianbao) - 版本发布说明 (Release Notes)

## Version 1.0.0 (2026-02-23)
**里程碑版本**: 惠电宝首个正式版本发布，集成了 RAG 检索、多模态识别与企业微信接入能力。

### 🎉 新增功能 (New Features)
- **RAG 混合检索引擎**: 上线基于 BM25 + Zhipu Embedding 的混合检索内核，大幅提升规程查询的召回率与相关性。
- **DeepSeek 推理接入**: 集成 DeepSeek-V3 (Chat) 与 R1 (Reasoner) 模型，实现智能路由，兼顾速度与深度。
- **企业微信全功能接入**: 支持文本、语音、图片消息的双向互通，并在回复前提供“正在思考”的状态反馈。
- **本地知识库构建**: 提供 Markdown 格式的知识库管理与自动化索引构建脚本 (`build_kb_index.py`)。
- **运维监控体系**: 上线 PM2 进程守护与 `system-monitor` 技能，支持系统资源实时监控。

### 🛠️ 优化与改进 (Improvements)
- **响应速度**: 优化了向量检索算法，平均检索耗时降低至 200ms 以内。
- **引用标注**: 重新设计了 Prompt 模板，强制要求输出内容必须包含明确的条款引用（如《安规》第X条）。
- **语音支持**: 增加了对 `edge-tts` 的支持，使得助手具备基础的语音回复能力。

### 🐛 修复问题 (Bug Fixes)
- 修复了 OpenClaw Gateway 在高并发下偶发的连接断开问题。
- 修复了知识库索引在特殊字符文件名下的解析错误。
- 解决了企业微信 Token 校验时因空格导致的签名失败问题。

### 📅 下一步计划 (Next Steps)
- 引入 Query Expansion (查询扩展) 技术，进一步提升模糊提问的理解能力。
- 增加对飞书/钉钉渠道的深度适配。
- 开发 Web 管理后台，支持可视化的知识库上传与管理。
