# 惠电宝 (Huidianbao) - 铁路电力智能安全助手

## 1. 产品概述 (Product Overview)
**惠电宝**是专为铁路电力专业设计的企业级智能助手。它基于先进的 OpenClaw 智能体框架，深度融合了大语言模型（LLM）与检索增强生成（RAG）技术，旨在解决一线电力作业人员面临的规程查询繁琐、作业风险识别困难、信息流转不畅等痛点。

作为您的“口袋里的安全专家”，惠电宝通过企业微信（WeCom）等即时通讯工具，提供7x24小时的规程咨询、故障诊断、多模态图像识别与作业辅助服务。

## 2. 核心价值 (Core Value)
- **权威合规**: 内置《高速铁路电力管理规则》、《铁路电力安全工作规程》等核心法规，确保每一次回答都有据可依。
- **即问即答**: 毫秒级响应速度，支持自然语言提问，无需记忆复杂的专业术语。
- **深度推理**: 不仅仅是搜索，更能基于 DeepSeek 推理模型进行故障分析与方案建议。
- **多模态感知**: 支持现场设备照片上传，自动识别设备缺陷与安全隐患。

## 3. 应用场景 (Use Cases)
### 3.1 规程查询与作业标准
- **场景**: 现场作业人员不确定“倒闸操作”的具体步骤。
- **交互**: 用户语音提问“倒闸操作的安全规定有哪些？”，惠电宝即刻反馈标准作业流程，并附带规程原文链接。

### 3.2 故障诊断与应急处理
- **场景**: 变电所出现异常告警，值班员需要快速判断原因。
- **交互**: 用户描述故障现象或上传报警截图，惠电宝结合知识库进行推理，给出可能的故障点及排查建议。

### 3.3 设备隐患识别
- **场景**: 巡检发现绝缘子疑似破损。
- **交互**: 拍照上传，惠电宝利用视觉模型分析图片，确认是否存在裂纹或放电痕迹。

## 4. 技术亮点 (Technical Highlights)
- **混合检索 (Hybrid Search)**: 结合 BM25 关键词匹配与 Vector Embedding 语义检索，确保专业术语与口语化提问均能精准召回。
- **模型路由 (Model Routing)**: 智能识别问题复杂度，简单问题走快速通道，复杂问题调用推理模型，平衡响应速度与深度。
- **数据安全**: 核心知识库本地化部署，敏感数据不离境，符合企业安全规范。
