# 惠电宝 - 运维与部署指南 (Operations & Deployment Guide)

## 1. 概述 (Overview)
本指南涵盖了惠电宝系统的部署、配置管理、日常运维、故障排查与应急恢复流程。系统以高可用、低维护成本为设计目标，通过自动化脚本与容器化技术简化运维复杂度。

## 2. 环境要求 (Prerequisites)
- **OS**: Linux (Ubuntu 22.04 LTS Recommended)
- **Python**: 3.10+ (Dependencies: `numpy`, `jieba`, `rank_bm25`, `zhipuai`, `requests`)
- **Node.js**: v18+ (For OpenClaw Runtime)
- **PM2**: 用于进程管理与日志监控。
- **Storage**: SSD (Recommended for fast RAG indexing).

## 3. 部署流程 (Deployment Steps)

### 3.1 基础环境准备
1. 安装 Node.js 与 PM2:
   ```bash
   curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
   sudo apt-get install -y nodejs
   npm install pm2 -g
   ```
2. 安装 Python 依赖:
   ```bash
   pip3 install -r requirements.txt
   ```

### 3.2 代码与配置同步
1. 克隆代码仓库:
   ```bash
   git clone https://github.com/huidianbao/openclaw-agent.git /root/.openclaw
   ```
2. 配置环境变量与 API Key:
   - 编辑 `/root/.openclaw/workspace/skills/hdb-rag/ask_rag.py` 中的 `API_KEY` (DeepSeek) 和 `ZHIPU_API_KEY` (Zhipu)。
   - 编辑 `/root/.openclaw/openclaw.json` 配置 IM 接入信息（如 WeCom Bot ID/Token）。

### 3.3 启动服务
使用 PM2 启动主服务:
```bash
cd /root/.openclaw
./start_hdb.sh
```
该脚本会自动检查并启动 `hdb-agent` 进程。

## 4. 日常运维 (Daily Operations)

### 4.1 服务监控
- 查看进程状态: `pm2 status`
- 查看实时日志: `pm2 logs hdb-agent --lines 50`
- 系统资源监控: `python3 /root/.openclaw/workspace/skills/system-monitor/monitor.py`

### 4.2 知识库维护
当 `/root/.openclaw/知识库资料` 目录下的 Markdown 文件发生变更（新增、修改、删除）时，必须执行索引重建：
```bash
python3 /root/.openclaw/build_kb_index.py
```
建议配置为 Cron Job，每日凌晨自动执行一次全量重建。

### 4.3 故障排查 (Troubleshooting)
- **RAG 检索无结果**:
  - 检查 `/root/.openclaw/kb_index.pkl` 是否存在且最新。
  - 检查 `build_kb_index.py` 执行日志是否有报错（如 `Request Entity Too Large`）。
- **IM 消息不回复**:
  - 检查网络连接与 DNS 解析。
  - 检查 PM2 日志中是否有 OpenClaw Gateway 报错。
  - 确认 API Key 余额是否充足。

## 5. 备份与恢复 (Backup & Recovery)
- **配置备份**: 定期备份 `/root/.openclaw/openclaw.json`。
- **知识库备份**: 定期备份 `/root/.openclaw/知识库资料` 目录。
- **索引重建**: 若索引文件损坏，删除 `.pkl` 文件并重新运行构建脚本即可。

## 6. 版本更新 (Upgrade)
1. 拉取最新代码: `git pull origin main`
2. 更新依赖: `npm install && pip3 install -r requirements.txt`
3. 重启服务: `pm2 restart hdb-agent`
